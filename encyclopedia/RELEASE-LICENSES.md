# Release Licenses And Notices

This app is an independent documentation and tooling project. It does not include EarthBound ROM bytes, extracted copyrighted assets, generated byte-equivalent source, generated SPC/WAV/PNG payloads, or any user-provided ROM.

## Project License

Choose the project license before publishing the GitHub repository. Good release candidates are:

- MIT for permissive app code and documentation.
- Apache-2.0 if explicit patent terms are desired.
- A split license if code and prose should have different terms.

Do not publish a public GitHub release without adding the chosen root `LICENSE` file.

## Third-Party Runtime Notices

Packaged Electron builds include Electron and its bundled runtime components. Include third-party notices for:

- Electron.
- Chromium.
- Node.js.
- Transitive packages bundled by the packager.

When `electron-builder` is installed, verify whether the generated package already includes the upstream license files. Keep this file as the app-level notice index and add any generated third-party notice bundle beside it when publishing binaries.

## EarthBound / Nintendo Notice

EarthBound and related names, characters, art, music, and ROM data are property of their respective rights holders. This project does not distribute those materials. Users must provide their own legally obtained ROM, and ROM-derived outputs are generated locally into the user's app-data workspace.

## Release Boundary

Checked into and packaged with the app:

- Electron app shell.
- Curated documentation, learning paths, source-safe ROM knowledge, schemas, and validators.
- Authored release catalog generated by `npm run build:content:authored`.

Generated only after ROM selection:

- Source scaffolds and source indexes.
- Graphics, sprites, maps, audio, previews, renderer outputs, and export ZIPs.
- Any ROM-derived cache files or local generated manifests.

Never distributed:

- ROM files.
- Raw extracted copyrighted assets.
- Generated byte-equivalent source.
- Generated audio/image payloads derived from the ROM.
